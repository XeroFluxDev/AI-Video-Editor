# Application Code

This folder will contain the AI Video Editor application.

## Planned Structure

```
app/
├── public/              # Web-accessible files
│   ├── index.php        # Main interface
│   └── assets/          # CSS, JS, images
├── api/                 # API endpoints
│   ├── video/           # Video operations
│   ├── ai/              # AI integration
│   └── subtitles/       # Subtitle operations
├── src/                 # PHP classes
│   └── FFmpegService.php
├── config/              # Configuration files
├── storage/             # File storage
│   ├── uploads/
│   ├── temp/
│   └── exports/
└── composer.json        # Dependencies
```
